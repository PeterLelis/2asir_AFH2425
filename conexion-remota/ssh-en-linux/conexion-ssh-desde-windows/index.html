Conexión SSH desde Windows

Para esta conexión se usará la herramienta PuTTY, que si no está intalada en en PC, se puede descarga e instalarla facilmente desde el siguiente enlace:
https://apps.microsoft.com/detail/xpfnzksklbp7rj?hl=es-ES&gl=ES

Tras abrir la herramineta PuTTY, se introduce la IP del equipo al que se desea conectar.

 

 

Obs.: el puerto debe ser el 22 y el tipo de conexión SSH.

 

 

La conexión ha sido establecida. Se puede ver el login as.
