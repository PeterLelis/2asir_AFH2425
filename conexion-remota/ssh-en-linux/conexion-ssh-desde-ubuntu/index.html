Conexión SSH desde Ubuntu

Se abre la terminal (ctrl + alt + t) y introduce el protocolo el siguiente comando:
ssh usuario@host
En este caso: peter@192.168.1.2

Pregunta:
¿Cómo se podría conectar un usuario sin tener que meter la contraseña cada vez que se conecta?
R.: La forma más segura de conectar un usuario por SSH sin tener que introducir la contraseña es mediante autenticación basada en pares de claves.

Conexión sin uso de clave

  1. Generar el par de claves desde el equipo cliente:
    ssh-keygen -t rsa -b 4096
  2. Copiar clave pública al servidor:
    ssh-copy-id peter@192.168.1.2
