SSH en Linux

Paso previos para la conexión SSH

Es importante que las máquinas estén dentro de la misma res, en este caso usaré la 192.168.1.0.

  1. Ubuntu1 - Servidor SSH: 192.168.1.2
  2. Ubunto2 - Cliente: 192.168.1.4
  3. Windows: 192.168.1.3
  • Crear usuarios que puedan acceder al equipo a través del ssh (adduser)

Asegurar que el firewall (ufw) permite la conexión SSHsudo adduser peterlelis

Instalar el Servidor SSH en Ubuntu

SSH permite establecer una conexión remota segura a la línea de comandos de un servidor. Para ello se usa el siguiente comando:

  • sudo ufw allow ssh

Instalación del paquete OpenSSH

  • sudo apt update
  • sudo apt upgrade -y
  • sudo apt install openssh-server -y

 Verificar el estado: 

  • sudo systemctl status ssh

En la imagen se puede ver que el servicio no está activo. Para activarlo usamos este comando:

  • sudo systemctl start ssh

Ahora es servicio está en marcha.

Ahora se habilita el servicio automático con el comando:

  • sudo systemctl enable ssh
