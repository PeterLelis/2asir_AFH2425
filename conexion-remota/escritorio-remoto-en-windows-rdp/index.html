 Escritorio Remoto en Windows (RDP)

Levantar servicio RDP en Windows

Antes de habilitar RDP, es necesario haber creado el usuario que va a usar la conexión remota:

Configuración / Cuentas / Familia y otros usuarios.
Clic en agregar otra persona a este equipo.

Pedirá la cuenta de Microsoft. Para crear una cuenta local hay que hacer clic en “no tengo los datos de esta persona” y luego crear cuenta local.

Tras crear el usuario hay que habilitar el Escritorio Remoto (RDP)

Se activa el Escritorio Remoto en Configuración / Sistema / Escritorio remoto. Confirmar.

Averiguar los permisos

En la misma ventana de Escritorio remoto, hacer clic en Usuarios que pueden acceder a este equipo en el apartado Cuenta de Usuario.

Probando conexión desde Windows Cliente al Windows Servidor

Abrir la herramienta de Conexión de Escritorio Remoto e introducir la IP del Windows Servidor. A continuación, pedirá el usuario y contraseña, creados en el paso anterior.

Saldrá un adivertencia de seguridad:

Conexión establecida

En la la parate superior de la imagen se puede ver que está conectado remotamente y la configuración de red en el CMD.

Conexión Linux Remmina

Para la conexión del Ubuntu cliente al Windows Servidor, es necesario instalar Remmina:
sudo apt update
sudo apt install remmina remmina-plugin-rdp

Se abre la ventana de Remmina, en la cual introducimos la IP del Windows Servidor y el protocolo RDP. A continuación, pedirá el usuario:
