TeamViwer

Instalación en Windows

Descarga para Windows: https://www.teamviewer.com/es/download/windows/

Tras descargar el archivo .exe desde el enlace arriba, se inicia el asistente de instalación haciedo doble clic en él. Seleccionar la instalación con los ajustes predeterminados:

Tras la instalación, es necesario aceptar los términos EULA y DPA para continuar. Luego la aplicación destará instalada y lista para su uso.

Instalación en Ubuntu

Descarga para Ubuntu: https://www.teamviewer.com/es/download/linux/

Se hace clic derecho en el archivo .deb descargado y seleccionar abrir con. Centro de Aplicaiones.

Dado que el paquete que estamos instalando es de tercero, hay un aviso de riegos. Clic en instalar.

La aplicación se ha instalado. Tal y como en Windows, hay que aceptar los términos EULA y DPA:

Conexión de Ubuntu a Windows

En la imagen abajo se ve la aplicaión en Ubuntu en la izquierda y en la derecha la aplicación en Windows:

Obs.: Para usar el acceso remoto en TeamViwer, es necesario crear una cuenta.

Tras iniciar sesión se introduce el ID del equipo al que se quiere conectar y su contraseña y la conexión se estabelecerá.

Conexión desde Windows a Ubuntu

Para llevar a cabo esta conexión, el proceso es el mismo.

Se introduce el ID del equipo al que se quiere conectar y su contraseña. La conexión se estabelecerá.

Conexión desde el móvil

Desde la aplicación del móvil se introduce el ID y luego la contraseña. La conexión se estabelecerá.
